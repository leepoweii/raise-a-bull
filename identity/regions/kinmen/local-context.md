## 文化遺產

### 風獅爺

- 金門最具代表性的文化符號，散布全島各聚落
- 功能為鎮風煞、擋路沖，不是裝飾品
- 每尊風獅爺造型不同，各有在地故事
- 居民至今仍會祭拜、披紅，是活的信仰而非觀光道具

### 閩南傳統建築與洋樓

- 閩南式古厝：燕尾脊、馬背、磚雕，以瓊林聚落與山后民俗文化村為代表
- 僑匯洋樓：清末至民初金門人赴南洋經商，匯錢回鄉蓋的中西合璧建築
- 代表洋樓：陳景蘭洋樓（金湖）、水頭得月樓（金城）
- 許多古厝已改為民宿，由金門國家公園管理的「傳統聚落民宿」品質穩定

### 戰地遺跡

- **翟山坑道**：花崗岩鑿出的地下水道，可容納小型登陸艇，金門最知名坑道
- **古寧頭戰史館**：1949 年古寧頭戰役紀念地
- **獅山砲陣地**：保留 8 吋榴砲，有砲操表演
- **北山斷崖 / 北山洋樓**：古寧頭戰役最激烈處，彈孔至今保留
- **馬山觀測站**：距廈門最近的據點，曾用於心戰喊話

> 戰地歷史是金門人的集體記憶，不是主題樂園。提及時保持尊重。

## 特產與飲食

| 品項 | 說明 |
|------|------|
| **高粱酒** | 金門酒廠（金酒）出品，58 度、38 度為主力。金門人日常就在喝，不只是伴手禮 |
| **貢糖** | 花生加麥芽糖，傳統零食。品牌多，各家口味略有差異 |
| **廣東粥** | 金門式早餐，粥底熬到米粒化開（糜粥），配油條。早上才有 |
| **牛肉麵** | 早年軍人帶來的飲食文化，金門牛肉多為本地溫體牛 |
| **蚵嗲** | 石蚵裹粉油炸，金門石蚵是野生潮間帶養殖，個頭小但味道濃 |
| **石蚵** | 金門特有的潮間帶養殖方式，用花崗岩石條讓蚵附著生長 |
| **一條根** | 在地藥草，做成貼布、藥膏，觀光客常買 |
| **麵線** | 手工日曬麵線，後浦一帶仍有製作 |

## 代表性生物

### 鱟（ㄏㄡˋ）

- 四億年前就存在的活化石，金門水頭與建功嶼一帶仍有野生族群
- 金門是台灣唯一還能看到鱟的地方
- 水試所有鱟的復育計畫

### 栗喉蜂虎

- 夏候鳥，每年 4-9 月來金門繁殖
- 在青年農莊、慈湖一帶的沙質崖壁築巢
- 羽色鮮豔（綠、藍、栗紅），是金門夏季代表鳥種
- 金門國家公園有設觀察站，有賞鳥季活動

### 歐亞水獺

- 金門是台灣唯一有野生歐亞水獺的地方
- 數量稀少（估計約 100-200 隻），常出沒於田間水道與水庫周邊
- 在地保育議題，開發與水獺棲地常有衝突

## 在地機構與資源

| 機構 | 角色 |
|------|------|
| **金門縣政府** | 地方行政機關 |
| **金門國家公園管理處** | 管轄戰地史蹟、傳統聚落、自然生態，金門約四分之一面積為國家公園範圍 |
| **金門縣文化局** | 文化資產保存、藝文活動 |
| **金門酒廠（金酒公司）** | 縣營事業，金門最大經濟支柱。酒廠收入佔縣府財源大宗 |
| **金門大學** | 在地唯一大學，有建築、閩南文化等特色科系 |
| **金門日報** | 在地報紙，了解金門時事的主要管道 |
